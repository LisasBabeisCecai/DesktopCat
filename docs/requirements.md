# 需求文档 — DesktopCat 桌面宠物

## 项目概述

为 macOS 桌面开发一个可交互的虚拟宠物小猫，使用 AI 生成的带透明通道视频作为皮肤，提供养护周期提醒功能。

## 功能需求

### F1. 桌面宠物显示

| 编号 | 需求 | 描述 |
|------|------|------|
| F1.1 | 透明窗口 | 小猫悬浮在桌面上，无边框、无背景、置顶显示 |
| F1.2 | 视频皮肤 | 支持 ProRes 4444 带 Alpha 通道的 MOV 视频 |
| F1.3 | 四种状态 | idle（待机）、walk（走路）、sit（坐下）、pet（被摸）|
| F1.4 | 所有空间可见 | 在所有 macOS 桌面空间（Space）中都显示 |
| F1.5 | 可拖动 | 按住小猫任意位置可拖动到屏幕任何地方 |

### F2. 行为状态机

| 编号 | 需求 | 描述 |
|------|------|------|
| F2.1 | 默认待机 | 启动后播放 idle 视频（循环），小猫坐着不动 |
| F2.2 | 自动散步 | 30 秒无互动 → walk 视频完整播放一遍（原地，不位移）|
| F2.3 | 坐下过渡 | walk 结束后紧接着播放 sit 视频一遍，然后回到 idle |
| F2.4 | 点击被摸 | 点击小猫 → pet 视频播放一遍 → 回到 idle |
| F2.5 | 不可打断 | walk/sit 播放期间点击无效 |
| F2.6 | 不位移 | 所有状态下窗口位置固定，小猫不会自动移动 |

### F3. 养护周期提醒

| 编号 | 需求 | 描述 |
|------|------|------|
| F3.1 | 6 项任务 | 剪指甲(15天)、掏耳朵(15天)、外驱(30天)、内驱(90天)、换猫砂(60天)、深度梳毛(7天) |
| F3.2 | 自动循环 | 标记完成后自动从当天算出下一次到期日 |
| F3.3 | 设置首次日期 | 可手动指定上次完成日期，系统从该日起算 |
| F3.4 | 提前 3 天提醒 | 到期前 3 天开始弹系统通知，每天 1 次 |
| F3.5 | 铃铛图标 | 有临期任务时小猫旁显示摇晃的🔔图标 |
| F3.6 | 逾期高亮 | 养护面板中逾期项红色、临期项橙色 |

### F4. 系统集成

| 编号 | 需求 | 描述 |
|------|------|------|
| F4.1 | 菜单栏图标 | 顶部菜单栏显示猫爪图标，点击展开菜单 |
| F4.2 | 不占 Dock | 程序不在 Dock 中显示图标（LSUIElement） |
| F4.3 | 数据持久化 | 养护任务存 SQLite，重启不丢失 |
| F4.4 | 系统通知 | 到期提醒使用 macOS 原生通知横幅 |

## 非功能需求

| 编号 | 需求 | 描述 |
|------|------|------|
| NF1 | 启动速度 | 启动到显示小猫 < 2 秒 |
| NF2 | CPU 占用 | 待机状态 CPU < 5%（单视频 30fps 解码）|
| NF3 | 内存占用 | < 200MB |
| NF4 | 系统兼容 | macOS 13.0+ (Ventura 及以上) |

## 视频素材规格

| 属性 | 要求 |
|------|------|
| 格式 | QuickTime (.mov) |
| 编码器 | Apple ProRes 4444 XQ |
| 像素格式 | yuva444p12le（带 Alpha） |
| 分辨率 | 1080×1080（程序内缩放到 200×200 显示）|
| 帧率 | 30fps |
| 背景 | Alpha 通道透明（背景像素 alpha=0）|
